macro_rules! src_file {
    ( $file_name:expr) => { ... };
}

Macro which returns the path to file in your src/ directory.

Example: src_file!("constants.rs") returns /path/to/project/src/constants.rs

If you need a more custom path, the basic implementation is: concat!(env!("CARGO_MANIFEST_DIR"), "/src/path/to/file")
